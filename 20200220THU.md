# 데이터베이스 변경 관리

1. **구조 변경(DDL)**
    - DDL의 특징 : **Auto commit**
    - 테이블(객체) 생성 : create
        한번 생성하면 변경할 수 없고 순서도 마찬가지
    - 테이블(객체) 삭제 : drop
    - 테이블(객체) 변경 : alter
    - 테이블 데이터 전체 삭제 : truncate

2. **데이터 변경(DML)**
    - 삽입 : insert
    - 수정 : update
    - 삭제 : delete
    - 병합 : merge

## 데이터타입

- varchar2(10), char(10)으로 만들어진 각 컬럼에 'a'입력
varchar2(10) 는 `'a'`
char(10)     는 'a 로 입력된다.
